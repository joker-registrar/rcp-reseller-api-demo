Property Value Description
Autorenew
{ERROR_INVALID_AR_OPT} {ERROR_INVALID_AR_VALUE}
autorenew on
autorenew off

 
Activate or deactivate the domain autorenewal system.
If activated, all domains which are marked to be autorenewed will be prolongued automatically 7 days before expiration. Notifications will be sent before and after the actual renewal.
In case autorenew is disabled, the standard renewal procedure will be in place again.
